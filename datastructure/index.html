Build data structure
Binary Search
            
  set left limit
  set right limit
                
  while left limit < right limit:
    pointer = (left + right) / 2 
                  
  if pointer > target:
    right limit = pointer
                
  else if pointer < target: 
    left limit = pointer
                
  else if pointer == target:
    return pointer
                
  return not found
Time complexity: O(log n)
Linear Search
  for element in array
  1.  if element = target
  2.    return target
  3.  else:
        continue
Time complexity: O(n)
Find Minimum
  1.  minimum value = first element of array
      for next element in array
  2.    if minimum value > next element
  3.      minimum value = next element
  4.    return minimum value
Time complexity: O(1)
Insert
  1.  insert number at the start
  2.  rebuild data structure
Time complexity: O(n)
Delete
  1.  search for element
  2.  remove element
  3.  shift divs to left
Time complexity: O(n)
